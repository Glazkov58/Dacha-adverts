## Дачник.ру ## (Dacha-advert)
Веб-приложение для размещения объявлений о продаже и аренде дачных участков, саженцев, инструментов и услуг. Платформа позволяет пользователям управлять своими объявлениями, добавлять фотографии, искать предложения по фильтрам и сохранять избранное.

 **Функционал**
Аутентификация и авторизация: Регистрация, вход, выход, защита личных данных.
Управление объявлениями: Создание, редактирование, удаление и просмотр объявлений (CRUD).
Загрузка фото: Возможность прикреплять до 5 фотографий к объявлению с автоматическим сохранением на сервере.
Поиск и фильтрация: Поиск по названию, фильтрация по категории и региону.
Избранное: Пользователи могут добавлять объявления в избранное.
Личный кабинет: Просмотр своих объявлений, статистика просмотров.
Адаптивный дизайн: Интерфейс построен на Bootstrap 5, корректно отображается на мобильных устройствах.
**Технологический стек**
Backend: Java 17+, Spring Boot 3, Spring Security, Spring Data JPA.
Database: MySQL 8.
Frontend: Thymeleaf, HTML5, CSS3, Bootstrap 5, JavaScript.
Security: BCrypt (хеширование паролей), Session Management.
Build Tool: Maven / Gradle.

 **Безопасность**
Пароли пользователей хранятся в хешированном виде (BCrypt).
Доступ к личному кабинету и управлению объявлениями защищен Spring Security.
Реализована защита от CSRF (отключена в текущей конфигурации для упрощения тестирования, без проблем можно включить для продакшана).
Валидация входных данных на стороне сервера и клиента.
